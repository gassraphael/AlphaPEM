AlphaPEM

PEM Fuel Cell Simulator

Welcome to AlphaPEM

This web interface provides simplified access to some of AlphaPEM’s features. AlphaPEM is an open-source PEM fuel cell simulator that rapidly models internal states and voltage dynamics across all operating conditions. It employs a 1D+1D (pseudo-2D) finite volume, physics-based model to perform dynamic, two-phase, non-isothermal simulations. The project is licensed under GNU GPL 3.0.

Available Simulations

Step Current

Simulates step current density changes and cell response.

Polarization Curve

Generates complete polarization curves across current range.

EIS Spectroscopy

Performs impedance spectroscopy analysis across frequencies.
