AlphaPEM

PEM Fuel Cell Simulator

Welcome to AlphaPEM

AlphaPEM is an open-source software package for simulating proton exchange membrane fuel cell (PEMFC) systems using physics-based models.

It quickly simulates the internal states and voltage dynamics of PEMFC systems for all current densities and operating conditions.

Available Simulations

Step Current

Simulates a step current density change and monitors the cell response, including voltage dynamics and internal states evolution.

Polarization Curve

Generates a complete polarization curve by sweeping current density, showing cell voltage as a function of current output.

EIS Spectroscopy

Performs electrochemical impedance spectroscopy analysis across a range of frequencies to characterize impedance response.

About AlphaPEM

  • Physics-based Model: 1D+1D pseudo-2D finite volume approach
  • Dynamic Simulation: Two-phase, non-isothermal with thermal effects
  • Configuration Support: Co-flow and counter-flow gas channels
  • Parameter Calibration: Optimize model to real fuel cell data
  • Open Source: Licensed under GNU GPL 3.0
